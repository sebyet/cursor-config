# Create Survey

Create and analyze user surveys to gather quantitative feedback and insights.

## What This Command Does

Creates surveys by:
- Designing survey questions
- Creating survey structure and flow
- Analyzing survey results
- Extracting insights from survey data
- Generating survey reports

## How It Works

1. **Define Survey Goals**
   - Ask about survey objectives
   - Identify research questions
   - Understand target audience
   - Determine survey type and length
   - Plan distribution method

2. **Design Survey**
   - Create survey questions
   - Design question types (multiple choice, rating, open-ended)
   - Structure survey flow and logic
   - Add skip logic and branching
   - Test survey structure

3. **Distribute and Collect**
   - Set up survey distribution
   - Plan participant recruitment
   - Monitor response rates
   - Collect survey responses
   - Track completion rates

4. **Analyze Results**
   - Review survey responses
   - Analyze quantitative data
   - Identify patterns and trends
   - Extract insights and findings
   - Generate survey analysis report

## What to Tell the User

- "What do you want to learn from this survey?"
- "What are your research questions?"
- "Who is your target audience?"
- Create and analyze survey
- "Survey analysis complete! Responses: [count]. Key findings: [list]. Insights: [list]"
