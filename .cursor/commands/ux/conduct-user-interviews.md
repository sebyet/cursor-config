# Conduct User Interviews

Conduct and analyze user interviews to gather qualitative insights and understand user needs.

## What This Command Does

Conducts user interviews by:
- Planning user interview sessions
- Creating interview guides and questions
- Conducting user interviews
- Analyzing interview transcripts
- Extracting insights and themes from interviews
- Generating interview reports

## How It Works

1. **Plan Interview Study**
   - Ask about research goals and questions
   - Identify target participants
   - Determine interview type (structured, semi-structured, unstructured)
   - Plan recruitment and scheduling
   - Define interview logistics

2. **Create Interview Materials**
   - Design interview guide and questions
   - Prepare follow-up questions
   - Create consent forms
   - Set up recording and note-taking
   - Prepare interview environment

3. **Conduct Interviews**
   - Facilitate interview sessions
   - Ask questions and probe deeper
   - Listen actively and take notes
   - Build rapport with participants
   - Document observations

4. **Analyze Interview Data**
   - Review interview transcripts and notes
   - Identify themes and patterns
   - Extract key insights and quotes
   - Categorize findings
   - Generate interview analysis report

## What to Tell the User

- "What do you want to learn from these interviews?"
- "What are your research questions?"
- "Who are your target participants?"
- Plan and conduct user interviews
- "Interview analysis complete! Key insights: [list]. Themes: [list]. Recommendations: [list]"
