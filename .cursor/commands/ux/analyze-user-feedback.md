# Analyze User Feedback

Synthesize user feedback from multiple sources to identify patterns and insights.

## What This Command Does

Analyzes user feedback by:
- Aggregating feedback from multiple sources
- Identifying themes and patterns
- Prioritizing issues by frequency/impact
- Generating actionable insights
- Creating summary reports

## How It Works

1. **Gather Feedback**
   - Ask for feedback sources (surveys, interviews, support tickets)
   - Collect feedback data
   - Understand context

2. **Analyze Patterns**
   - Identify common themes
   - Categorize feedback types
   - Prioritize by frequency and impact
   - Find root causes

3. **Generate Insights**
   - Create summary of findings
   - Highlight top issues
   - Suggest actionable improvements
   - Prioritize recommendations

## What to Tell the User

- "What feedback sources do you have? (surveys, interviews, support tickets, etc.)"
- Analyze feedback and identify patterns
- Generate insights report
- "Analysis complete! Top insights: [list]. Recommended actions: [list]"
