# Create Persona

Generate user personas based on research data and user insights.

## What This Command Does

Creates detailed user personas that include:
- Demographics and background
- Goals and motivations
- Pain points and frustrations
- Behaviors and preferences
- Use cases and scenarios
- Quotes and characteristics

## How It Works

1. **Gather User Data**
   - Ask about user research findings
   - Understand user segments
   - Get insights from interviews/surveys

2. **Create Persona**
   - Define demographics
   - Identify goals and motivations
   - Document pain points
   - Describe behaviors
   - Add quotes and characteristics

3. **Generate Persona Document**
   - Create well-structured persona
   - Include visual elements if possible
   - Make it actionable for design/dev

## What to Tell the User

- "What user segment do you want to create a persona for?"
- "What research data do you have about these users?"
- Generate persona document
- "Persona created! Use this to guide design and development decisions."
