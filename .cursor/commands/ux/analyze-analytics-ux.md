# Analyze Analytics UX

Analyze analytics data from a UX perspective to identify user experience issues and opportunities.

## What This Command Does

Analyzes analytics from UX perspective by:
- Interpreting analytics data for UX insights
- Identifying UX issues from analytics
- Analyzing user behavior patterns
- Finding conversion barriers
- Generating UX-focused analytics reports

## How It Works

1. **Understand Analytics Context**
   - Ask about analytics platform and data
   - Identify UX questions to answer
   - Understand user goals and tasks
   - Get context on key user flows
   - Determine analysis scope

2. **Analyze User Behavior**
   - Review user behavior patterns
   - Identify drop-off points and friction
   - Analyze task completion rates
   - Find navigation issues
   - Identify user flow problems

3. **Identify UX Issues**
   - Find conversion barriers
   - Identify usability problems
   - Note confusing or problematic areas
   - Find content or navigation issues
   - Identify user frustration points

4. **Generate UX Insights**
   - Create UX-focused analysis
   - Prioritize UX issues
   - Suggest UX improvements
   - Generate actionable recommendations
   - Create UX analytics report

## What to Tell the User

- "What analytics data do you have?"
- "What UX questions do you want to answer?"
- "What are the key user flows to analyze?"
- Analyze analytics from UX perspective
- "UX analytics analysis complete! Issues found: [list]. Insights: [list]. Recommendations: [list]"
