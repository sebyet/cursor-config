# Create User Journey

Map user journeys to visualize the complete user experience from start to finish.

## What This Command Does

Creates user journey maps that include:
- User persona and scenario
- Journey stages (awareness, consideration, purchase, etc.)
- Touchpoints and interactions
- User emotions and pain points
- Opportunities for improvement
- Metrics at each stage

## How It Works

1. **Define Journey Scope**
   - Ask about the user journey to map
   - Identify starting and ending points
   - Understand the user persona
   - Define the scenario

2. **Map Journey Stages**
   - Break down into stages
   - Identify touchpoints
   - Document user actions
   - Note emotions and pain points

3. **Generate Journey Map**
   - Create visual journey map
   - Highlight pain points
   - Identify opportunities
   - Suggest improvements

## What to Tell the User

- "What user journey do you want to map?"
- "Who is the user? What's the scenario?"
- "What are the key stages?"
- Generate user journey map
- "User journey mapped! Key pain points: [list]. Opportunities: [list]"
