# Test Cross-Browser

Cross-browser testing checklist to ensure compatibility across different browsers.

## What This Command Does

Tests cross-browser compatibility by:
- Identifying browsers to test
- Creating test checklist
- Verifying functionality across browsers
- Identifying browser-specific issues
- Generating compatibility report

## How It Works

1. **Define Test Scope**
   - Ask about browsers to test
   - Identify target browsers
   - Understand test requirements
   - Get test scenarios

2. **Create Test Checklist**
   - List browsers to test
   - Define test scenarios
   - Create test checklist
   - Plan test execution

3. **Generate Test Report**
   - Document test results
   - List browser-specific issues
   - Prioritize problems
   - Recommend fixes

## What to Tell the User

- "What browsers do you want to test?"
- "What functionality needs testing?"
- Create cross-browser test checklist
- "Cross-browser test checklist created! Test these browsers: [list]. Issues found: [list]"
