# Test Accessibility

Accessibility testing to ensure applications are usable by people with disabilities.

## What This Command Does

Tests accessibility by:
- Checking WCAG compliance
- Testing keyboard navigation
- Verifying screen reader compatibility
- Checking color contrast
- Identifying accessibility issues

## How It Works

1. **Define Test Scope**
   - Ask what to test
   - Identify accessibility standards (WCAG 2.1, etc.)
   - Understand test requirements
   - Get access to application

2. **Perform Accessibility Tests**
   - Check WCAG compliance
   - Test keyboard navigation
   - Verify screen reader support
   - Check color contrast
   - Identify issues

3. **Generate Test Report**
   - Create accessibility report
   - List issues found
   - Prioritize by severity
   - Recommend fixes

## What to Tell the User

- "What do you want to test for accessibility?"
- "What accessibility standards? (WCAG 2.1, etc.)"
- Perform accessibility testing
- "Accessibility test complete! Issues found: [count]. Critical: [list]. Recommendations: [list]"
