# E2E Test

Create end-to-end tests for user flows and critical paths using Playwright or Cypress.

## What This Command Does

Creates E2E tests for:
- User flow testing
- Critical path validation
- Authentication flows
- Form submissions
- Navigation flows
- Error scenarios

## How It Works

1. **Understand E2E Testing Needs**
   - Ask about user flows to test
   - Identify critical paths
   - Understand test scenarios
   - Get context on user journeys

2. **Set Up E2E Framework**
   - Choose test framework (Playwright recommended)
   - Install dependencies
   - Configure test environment
   - Set up test utilities
   - Configure test data management

3. **Create Test Plan**
   - Define test scenarios
   - Identify test cases
   - Plan test data
   - Document test plan

4. **Write E2E Tests**
   - Write user flow tests
   - Create critical path tests
   - Add authentication tests
   - Test form submissions
   - Test navigation flows
   - Add error scenario tests

5. **Run and Verify Tests**
   - Run E2E tests
   - Verify test coverage
   - Check test reliability
   - Fix any test issues
   - Document test results

## What to Tell the User

- "What user flows do you want to test?"
- "What are your critical paths?"
- Create E2E tests
- "E2E tests created! Framework: [name]. Test scenarios: [count]. Run tests with: [command]"
