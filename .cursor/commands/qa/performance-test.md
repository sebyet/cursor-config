# Performance Test

Create comprehensive performance tests including scenarios, benchmarks, and regression testing.

## What This Command Does

Creates performance tests for:
- Performance test scenarios
- Performance benchmarks
- Performance regression testing
- Load testing scenarios
- Stress testing scenarios
- Performance monitoring

## How It Works

1. **Understand Performance Testing Needs**
   - Ask about performance goals
   - Identify performance metrics
   - Understand load requirements
   - Get context on requirements

2. **Create Test Plan**
   - Define performance scenarios
   - Set performance benchmarks
   - Plan load testing
   - Set up test environment
   - Document test plan

3. **Set Up Performance Testing**
   - Configure performance tools
   - Set up test scenarios
   - Configure load patterns
   - Set up monitoring
   - Configure test data

4. **Create Performance Tests**
   - Write performance scenarios
   - Create load tests
   - Add stress tests
   - Test performance limits
   - Validate performance metrics

5. **Run and Analyze Tests**
   - Run performance tests
   - Analyze performance data
   - Compare against benchmarks
   - Identify performance issues
   - Generate performance report

## What to Tell the User

- "What performance testing do you need?"
- "What are your performance goals?"
- Create performance tests
- "Performance tests created! Scenarios: [count]. Benchmarks: [set]. Run tests with: [command]"
