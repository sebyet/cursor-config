# Write Test Cases

Create detailed test cases with steps, expected results, and test data.

## What This Command Does

Creates test cases that include:
- Test case ID and description
- Preconditions and setup
- Test steps
- Expected results
- Test data requirements

## How It Works

1. **Understand Feature/Functionality**
   - Ask about feature to test
   - Understand requirements
   - Identify test scenarios
   - Get acceptance criteria

2. **Create Test Cases**
   - Write test case descriptions
   - Define test steps
   - Specify expected results
   - Add test data requirements

3. **Generate Test Cases**
   - Create structured test cases
   - Include all necessary details
   - Make them executable
   - Ready for testing

## What to Tell the User

- "What feature do you want test cases for?"
- "What are the requirements and acceptance criteria?"
- Generate test cases
- "Test cases created! [count] test cases ready for execution."
