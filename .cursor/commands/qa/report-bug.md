# Report Bug

Generate bug reports with all necessary information for developers to reproduce and fix.

## What This Command Does

Creates bug reports that include:
- Bug title and description
- Steps to reproduce
- Expected vs actual behavior
- Environment details
- Screenshots/logs if available
- Priority and severity

## How It Works

1. **Gather Bug Information**
   - Ask about the bug
   - Get steps to reproduce
   - Understand expected behavior
   - Collect environment details

2. **Create Bug Report**
   - Write clear description
   - Document reproduction steps
   - Note expected vs actual
   - Add environment info
   - Assign priority/severity

3. **Generate Bug Report**
   - Create structured report
   - Include all necessary details
   - Make it actionable
   - Ready for development

## What to Tell the User

- "What bug did you find?"
- "Can you describe what happened?"
- "What were the steps to reproduce?"
- Generate bug report
- "Bug report created! Ready for development team."
