# Manage Infrastructure

Manage infrastructure lifecycle - provision, update, and deprovision resources.

## What This Command Does

Manages infrastructure by:
- Provisioning infrastructure resources
- Updating infrastructure configurations
- Deprovisioning unused resources
- Managing infrastructure state
- Documenting changes

## How It Works

1. **Understand Infrastructure Needs**
   - Ask about infrastructure action
   - Identify resources to manage
   - Understand change scope
   - Determine management approach

2. **Manage Infrastructure**
   - Provision resources
   - Update configurations
   - Deprovision resources
   - Manage state

3. **Document and Verify**
   - Document changes
   - Verify infrastructure
   - Update state
   - Plan next steps

## What to Tell the User

- "What infrastructure action? (provision, update, deprovision)"
- "What resources should we manage?"
- Manage infrastructure and verify
- "Infrastructure management complete! Action: [type]. Resources: [managed]. State: [updated]"
