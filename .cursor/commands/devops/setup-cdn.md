# Setup CDN

Configure Content Delivery Network (CDN) for improved performance and global distribution.

## What This Command Does

Sets up CDN by:
- Configuring CDN provider (Cloudflare, AWS CloudFront, etc.)
- Setting up caching rules
- Configuring edge locations
- Setting up SSL/TLS
- Optimizing content delivery

## How It Works

1. **Understand CDN Needs**
   - Ask about CDN provider
   - Identify content to cache
   - Understand geographic requirements
   - Determine caching requirements

2. **Design CDN Setup**
   - Choose CDN provider
   - Configure caching rules
   - Set up edge locations
   - Plan SSL/TLS

3. **Generate Configuration**
   - Create CDN configuration
   - Set up caching
   - Configure SSL/TLS
   - Document setup steps

## What to Tell the User

- "What CDN provider are you using? (Cloudflare, AWS CloudFront, etc.)"
- "What content should we cache?"
- Generate CDN configuration
- "CDN setup complete! Provider: [name]. Caching: [configured]. Edge locations: [count]"
