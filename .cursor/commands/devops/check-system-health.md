# Check System Health

Check system health and performance across applications and infrastructure.

## What This Command Does

Checks system health by:
- Reviewing key metrics
- Identifying performance issues
- Checking error rates
- Assessing resource utilization
- Generating health report

## How It Works

1. **Define Health Checks**
   - Ask what systems to check
   - Identify key metrics
   - Understand health criteria
   - Get access to monitoring

2. **Perform Health Check**
   - Review metrics
   - Check error rates
   - Assess performance
   - Identify issues

3. **Generate Health Report**
   - Create status report
   - Highlight issues
   - Prioritize problems
   - Recommend actions

## What to Tell the User

- "What systems do you want to check?"
- "What are your health criteria?"
- Perform health check
- "System health check complete! Status: [summary]. Issues: [list]. Recommendations: [list]"
