# Setup Monitoring

Configure monitoring and alerts for applications, infrastructure, and services.

## What This Command Does

Sets up monitoring by:
- Configuring monitoring tools
- Setting up alerts and thresholds
- Creating dashboards
- Defining SLOs and SLIs
- Establishing alerting rules

## How It Works

1. **Understand Monitoring Needs**
   - Ask about what to monitor
   - Identify key metrics
   - Understand infrastructure
   - Determine alerting requirements

2. **Design Monitoring Setup**
   - Choose monitoring tools
   - Define metrics to track
   - Set up alert thresholds
   - Create dashboards

3. **Generate Configuration**
   - Create monitoring config
   - Set up alerts
   - Document setup steps
   - Provide implementation guide

## What to Tell the User

- "What do you want to monitor? (applications, infrastructure, services)"
- "What are your key metrics and SLOs?"
- Generate monitoring configuration
- "Monitoring setup complete! Configured: [list]. Alerts: [list]"
