# Setup APM

Configure Application Performance Monitoring (New Relic, Datadog APM, etc.).

## What This Command Does

Sets up APM by:
- Configuring APM tool (New Relic, Datadog APM, etc.)
- Setting up application instrumentation
- Creating performance dashboards
- Establishing performance baselines
- Configuring performance alerts

## How It Works

1. **Understand APM Needs**
   - Ask about APM tool
   - Identify applications to monitor
   - Understand performance requirements
   - Determine alerting thresholds

2. **Design APM Setup**
   - Choose APM tool
   - Configure instrumentation
   - Set up dashboards
   - Define performance baselines

3. **Generate Configuration**
   - Create APM configuration
   - Set up instrumentation
   - Configure alerts
   - Document setup steps

## What to Tell the User

- "What APM tool are you using? (New Relic, Datadog APM, etc.)"
- "What applications should we monitor?"
- Generate APM configuration
- "APM setup complete! Tool: [name]. Dashboards: [list]. Alerts: [configured]"
