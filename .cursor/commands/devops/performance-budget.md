# Performance Budget

Set and monitor performance budgets to ensure applications meet performance goals.

## What This Command Does

Manages performance budgets by:
- Setting performance budgets (LCP, FID, CLS, bundle size, etc.)
- Monitoring budget compliance
- Alerting on budget violations
- Generating performance reports
- Recommending optimizations

## How It Works

1. **Understand Performance Budget Needs**
   - Ask about performance goals
   - Identify metrics to budget
   - Understand current performance
   - Determine budget thresholds

2. **Set Performance Budgets**
   - Define budget metrics
   - Set budget thresholds
   - Configure monitoring
   - Plan alerting

3. **Monitor and Report**
   - Monitor budget compliance
   - Alert on violations
   - Generate reports
   - Recommend optimizations

## What to Tell the User

- "What performance metrics should we budget? (LCP, FID, CLS, bundle size, etc.)"
- "What are your performance goals?"
- Set performance budgets and configure monitoring
- "Performance budget setup complete! Metrics: [list]. Thresholds: [set]. Monitoring: [configured]"
