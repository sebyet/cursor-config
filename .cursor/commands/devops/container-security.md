# Container Security

Scan container images for vulnerabilities and security issues.

## What This Command Does

Scans containers by:
- Scanning container images for vulnerabilities
- Checking base image security
- Auditing container configurations
- Identifying security best practices
- Generating security reports

## How It Works

1. **Understand Container Security Needs**
   - Ask about container images
   - Identify scanning tools
   - Understand security requirements
   - Determine scan scope

2. **Scan Containers**
   - Scan images for vulnerabilities
   - Check configurations
   - Audit security practices
   - Identify issues

3. **Generate Report**
   - Create security report
   - List vulnerabilities
   - Provide recommendations
   - Document findings

## What to Tell the User

- "What container images should we scan?"
- "What scanning tool are you using?"
- Scan containers and generate report
- "Container security scan complete! Images: [count]. Vulnerabilities: [list]. Recommendations: [list]"
