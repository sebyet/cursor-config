# Manage DNS

Manage DNS records and configurations for domain routing and resolution.

## What This Command Does

Manages DNS by:
- Configuring DNS records
- Setting up DNS zones
- Configuring DNS routing
- Setting up DNS failover
- Managing DNS security

## How It Works

1. **Understand DNS Requirements**
   - Ask about domain and DNS needs
   - Identify DNS records to configure
   - Understand routing requirements
   - Determine DNS provider
   - Get context on existing DNS setup

2. **Configure DNS Records**
   - Create A, AAAA, CNAME, MX records
   - Configure subdomain routing
   - Set up DNS aliases
   - Configure DNS TTL values
   - Document DNS configuration

3. **Set Up DNS Routing and Failover**
   - Configure DNS routing policies
   - Set up DNS failover
   - Configure health checks
   - Set up geographic routing (if needed)
   - Document routing configuration

4. **Configure DNS Security**
   - Set up DNSSEC
   - Configure DNS filtering
   - Set up DNS monitoring
   - Configure DNS access controls
   - Document security configuration

## What to Tell the User

- "What domain needs DNS configuration?"
- "What DNS records need to be created?"
- "What DNS provider are you using?"
- Configure DNS records and routing
- "DNS management complete! Records: [created]. Routing: [configured]. Security: [setup]"
