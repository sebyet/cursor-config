# Data Retention Policy

Set up data retention and deletion policies for compliance and privacy.

## What This Command Does

Sets up data retention by:
- Defining retention periods
- Configuring automatic deletion
- Setting up data archival
- Ensuring compliance (GDPR, etc.)
- Documenting retention policies

## How It Works

1. **Understand Retention Needs**
   - Ask about compliance requirements
   - Identify data types
   - Understand retention periods
   - Determine deletion policies

2. **Design Retention Policy**
   - Define retention periods
   - Configure deletion
   - Plan archival
   - Ensure compliance

3. **Generate Configuration**
   - Create retention policy
   - Set up automation
   - Configure compliance
   - Document policy

## What to Tell the User

- "What compliance requirements do you need? (GDPR, etc.)"
- "What data types need retention policies?"
- Generate data retention policy
- "Data retention policy setup complete! Types: [count]. Retention: [periods]. Compliance: [requirements]"
