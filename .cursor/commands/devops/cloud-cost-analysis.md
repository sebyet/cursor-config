# Cloud Cost Analysis

Analyze cloud costs and generate detailed reports for cost management and optimization.

## What This Command Does

Analyzes cloud costs by:
- Analyzing cloud cost data
- Generating cost reports
- Identifying cost trends
- Allocating costs by team/project
- Generating cost optimization recommendations

## How It Works

1. **Gather Cost Data**
   - Ask about cloud provider and cost data
   - Identify cost analysis period
   - Understand cost allocation needs
   - Determine report requirements
   - Get access to cost data

2. **Analyze Cloud Costs**
   - Analyze cost by service
   - Analyze cost by resource
   - Identify cost trends
   - Compare cost periods
   - Find cost anomalies

3. **Generate Cost Reports**
   - Create cost breakdown reports
   - Generate cost allocation reports
   - Create cost trend reports
   - Generate cost forecast
   - Document cost analysis

4. **Provide Recommendations**
   - Identify cost optimization opportunities
   - Recommend cost-saving actions
   - Prioritize recommendations
   - Calculate potential savings
   - Create action plan

## What to Tell the User

- "What cloud provider are you using?"
- "What cost data do you have?"
- "What's your cost analysis period?"
- Analyze cloud costs and generate reports
- "Cloud cost analysis complete! Total cost: [amount]. Trends: [identified]. Recommendations: [list]"
