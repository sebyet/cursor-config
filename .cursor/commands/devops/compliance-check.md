# Compliance Check

Check compliance with regulations like GDPR, SOC2, HIPAA, and other standards.

## What This Command Does

Checks compliance by:
- Reviewing data handling practices
- Auditing security controls
- Checking privacy policies
- Verifying regulatory requirements
- Generating compliance report

## How It Works

1. **Identify Compliance Requirements**
   - Ask about regulations to check
   - Understand compliance scope
   - Identify requirements
   - Get compliance framework

2. **Perform Compliance Check**
   - Review data practices
   - Audit security controls
   - Check privacy policies
   - Verify requirements

3. **Generate Compliance Report**
   - Create compliance report
   - List gaps found
   - Recommend improvements
   - Provide remediation steps

## What to Tell the User

- "What compliance standards do you need to check? (GDPR, SOC2, HIPAA, etc.)"
- "What's your compliance scope?"
- Perform compliance check
- "Compliance check complete! Status: [compliant/non-compliant]. Gaps: [list]. Recommendations: [list]"
