# Create Runbook

Generate runbooks for common issues, incidents, and operational procedures.

## What This Command Does

Creates runbooks that include:
- Problem identification steps
- Troubleshooting procedures
- Resolution steps
- Escalation paths
- Prevention measures

## How It Works

1. **Understand Issue/Procedure**
   - Ask about the issue or procedure
   - Identify common scenarios
   - Understand resolution steps
   - Get context on system

2. **Create Runbook**
   - Document problem identification
   - Write troubleshooting steps
   - Define resolution procedures
   - Add escalation paths

3. **Generate Runbook Document**
   - Create structured runbook
   - Include all necessary steps
   - Add troubleshooting tips
   - Make it actionable

## What to Tell the User

- "What issue or procedure do you want a runbook for?"
- "What are the common scenarios?"
- Generate runbook
- "Runbook created! Ready for operations team."
