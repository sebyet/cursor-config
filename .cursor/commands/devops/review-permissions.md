# Review Permissions

Review access controls and permissions to ensure proper security and least privilege.

## What This Command Does

Reviews permissions by:
- Auditing user access controls
- Checking role-based permissions
- Verifying least privilege principle
- Identifying excessive permissions
- Recommending improvements

## How It Works

1. **Understand Permission Structure**
   - Ask about permission system
   - Identify roles and users
   - Understand access requirements
   - Get permission data

2. **Review Permissions**
   - Audit user permissions
   - Check role assignments
   - Verify least privilege
   - Identify issues

3. **Generate Review Report**
   - List permission issues
   - Recommend changes
   - Suggest improvements
   - Provide remediation steps

## What to Tell the User

- "What permission system are you using?"
- "What roles and users need review?"
- Review permissions
- "Permission review complete! Issues: [list]. Recommendations: [list]"
