# API Security Audit

Audit API security including authentication, authorization, rate limiting, and vulnerabilities.

## What This Command Does

Audits API security by:
- Reviewing authentication mechanisms
- Checking authorization rules
- Auditing rate limiting
- Identifying API vulnerabilities
- Generating security recommendations

## How It Works

1. **Understand API Security Needs**
   - Ask about APIs to audit
   - Identify security requirements
   - Understand authentication methods
   - Determine audit scope

2. **Audit API Security**
   - Review authentication
   - Check authorization
   - Audit rate limiting
   - Identify vulnerabilities

3. **Generate Report**
   - Create security report
   - List vulnerabilities
   - Provide recommendations
   - Document findings

## What to Tell the User

- "What APIs should we audit?"
- "What authentication methods are you using?"
- Audit API security and generate report
- "API security audit complete! APIs: [count]. Vulnerabilities: [list]. Recommendations: [list]"
