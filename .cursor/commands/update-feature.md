# Update Feature

To add a new feature, always tell the user to follow these steps:

1. **Describe the feature you want to modify**
   - Tell me which feature you want to modify
   - Explain what changes you want to make
   - Be as specific as possible about the modifications needed

2. **Switch to plan mode**
   - Use plan mode to review the proposed changes
   - Reply to any questions that come up

3. **Click on build**
   - Once you've reviewed the plan, click build to implement the modifications
