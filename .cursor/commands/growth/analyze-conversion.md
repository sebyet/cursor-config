# Analyze Conversion

Analyze conversion funnels to identify drop-off points and optimization opportunities.

## What This Command Does

Analyzes conversion by:
- Mapping conversion funnels
- Identifying drop-off points
- Calculating conversion rates
- Finding optimization opportunities
- Generating recommendations

## How It Works

1. **Understand Conversion Funnel**
   - Ask about conversion funnel
   - Identify funnel stages
   - Understand user journey
   - Get conversion data

2. **Analyze Funnel**
   - Map conversion stages
   - Calculate conversion rates
   - Identify drop-off points
   - Find bottlenecks

3. **Generate Analysis**
   - Create funnel analysis
   - Highlight drop-off points
   - Identify opportunities
   - Recommend optimizations

## What to Tell the User

- "What conversion funnel do you want to analyze?"
- "What are the funnel stages?"
- Analyze conversion funnel
- "Conversion analysis complete! Drop-off points: [list]. Conversion rates: [details]. Recommendations: [list]"
