# Event Tracking

Set up custom event tracking for analytics, user behavior, and conversion tracking.

## What This Command Does

Sets up event tracking by:
- Creating event schema
- Configuring event tracking
- Integrating analytics
- Setting up event triggers
- Tracking custom events
- Analyzing event data

## How It Works

1. **Understand Tracking Needs**
   - Ask about events to track
   - Identify tracking goals
   - Understand analytics needs
   - Get context on events

2. **Design Event Schema**
   - Define event structure
   - Create event taxonomy
   - Set up event properties
   - Document event schema

3. **Set Up Event Tracking**
   - Implement event tracking
   - Configure event triggers
   - Add event properties
   - Set up event validation
   - Test event tracking

4. **Integrate Analytics**
   - Connect analytics tools
   - Configure event mapping
   - Set up event funnels
   - Create event dashboards
   - Document tracking setup

## What to Tell the User

- "What events do you want to track?"
- "What analytics tool are you using?"
- Set up event tracking
- "Event tracking setup complete! Events: [count]. Analytics: [integrated]. Dashboard: [url]"
