# Cohort Analysis

Perform user cohort analysis to understand retention, behavior, and lifecycle patterns.

## What This Command Does

Analyzes cohorts by:
- Segmenting users into cohorts
- Analyzing retention patterns
- Tracking cohort behavior
- Measuring cohort metrics
- Identifying cohort trends
- Generating cohort insights

## How It Works

1. **Understand Cohort Needs**
   - Ask about cohort analysis goals
   - Identify cohort dimensions
   - Understand time periods
   - Get context on metrics

2. **Segment Cohorts**
   - Define cohort criteria
   - Create cohort segments
   - Set up cohort tracking
   - Configure cohort data

3. **Analyze Cohorts**
   - Calculate retention rates
   - Analyze cohort behavior
   - Track cohort metrics
   - Identify trends
   - Compare cohorts

4. **Generate Insights**
   - Identify patterns
   - Find insights
   - Generate recommendations
   - Create cohort reports
   - Visualize cohort data

## What to Tell the User

- "What cohort analysis do you want to perform?"
- "What cohort dimensions are important?"
- Analyze cohorts
- "Cohort analysis complete! Cohorts: [count]. Retention insights: [details]. Recommendations: [list]"
