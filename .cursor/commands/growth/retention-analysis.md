# Retention Analysis

Analyze user retention metrics, churn patterns, and retention strategies.

## What This Command Does

Analyzes retention by:
- Calculating retention metrics
- Analyzing churn patterns
- Identifying retention drivers
- Tracking retention trends
- Generating retention insights
- Recommending retention strategies

## How It Works

1. **Understand Retention Goals**
   - Ask about retention objectives
   - Identify retention metrics
   - Understand time periods
   - Get context on goals

2. **Calculate Retention Metrics**
   - Calculate retention rates
   - Analyze retention cohorts
   - Track retention trends
   - Compare retention periods
   - Identify retention patterns

3. **Analyze Churn**
   - Identify churn patterns
   - Analyze churn reasons
   - Find churn triggers
   - Compare churn rates
   - Generate churn insights

4. **Generate Retention Strategy**
   - Identify retention drivers
   - Recommend retention tactics
   - Create retention plan
   - Set retention goals
   - Generate retention report

## What to Tell the User

- "What retention analysis do you want to perform?"
- "What retention metrics are important?"
- Analyze retention
- "Retention analysis complete! Retention rate: [percentage]. Churn insights: [details]. Recommendations: [list]"
