# Optimize Performance for SEO

Optimize page performance to improve SEO rankings and user experience.

## What This Command Does

Optimizes performance for SEO by:
- Analyzing page speed
- Identifying performance issues
- Recommending optimizations
- Improving Core Web Vitals
- Creating optimization plan

## How It Works

1. **Understand Performance Context**
   - Ask about pages to optimize
   - Get current performance data
   - Understand performance goals
   - Identify Core Web Vitals

2. **Analyze Performance**
   - Review page speed metrics
   - Check Core Web Vitals
   - Identify bottlenecks
   - Find optimization opportunities

3. **Generate Optimization Plan**
   - Create recommendations
   - Prioritize improvements
   - Suggest specific optimizations
   - Provide implementation steps

## What to Tell the User

- "What pages do you want to optimize for performance?"
- "What are your current performance metrics?"
- Analyze and generate optimization plan
- "Performance optimization complete! Issues: [list]. Recommendations: [list]. Expected improvements: [details]"
