# Generate Meta Tags

Generate optimized meta tags including title, description, and Open Graph tags.

## What This Command Does

Generates meta tags by:
- Creating SEO-optimized titles
- Writing compelling descriptions
- Generating Open Graph tags
- Adding Twitter Card tags
- Ensuring proper formatting

## How It Works

1. **Understand Page Context**
   - Ask about page to optimize
   - Identify target keywords
   - Understand page purpose
   - Get content context

2. **Generate Meta Tags**
   - Create optimized title
   - Write description
   - Generate Open Graph tags
   - Add Twitter Card tags

3. **Review and Refine**
   - Check length requirements
   - Verify keyword usage
   - Ensure compelling copy
   - Make final adjustments

## What to Tell the User

- "What page do you want meta tags for?"
- "What are your target keywords?"
- Generate meta tags
- "Meta tags generated! Title: [title]. Description: [description]. Ready to implement."
