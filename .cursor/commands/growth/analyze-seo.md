# Analyze SEO

Analyze SEO performance including rankings, traffic, and optimization opportunities.

## What This Command Does

Analyzes SEO by:
- Reviewing search rankings
- Analyzing organic traffic
- Identifying top keywords
- Finding optimization opportunities
- Generating performance report

## How It Works

1. **Gather SEO Data**
   - Ask about SEO data sources
   - Get search console data
   - Understand time period
   - Identify key metrics

2. **Analyze Performance**
   - Review rankings
   - Analyze traffic trends
   - Identify top keywords
   - Find opportunities

3. **Generate Analysis Report**
   - Create performance report
   - Highlight key insights
   - Identify opportunities
   - Recommend actions

## What to Tell the User

- "What SEO data do you have? (Search Console, analytics, etc.)"
- "What time period to analyze?"
- Analyze SEO performance
- "SEO analysis complete! Rankings: [summary]. Traffic: [trends]. Opportunities: [list]"
