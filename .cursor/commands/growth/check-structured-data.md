# Check Structured Data

Validate structured data (JSON-LD, Schema.org) for proper implementation and SEO benefits.

## What This Command Does

Checks structured data by:
- Validating schema markup
- Checking for errors
- Verifying implementation
- Identifying missing schemas
- Recommending improvements

## How It Works

1. **Understand Structured Data**
   - Ask about pages to check
   - Identify schema types needed
   - Get current structured data
   - Understand SEO goals

2. **Validate Structured Data**
   - Check schema markup
   - Validate syntax
   - Verify implementation
   - Identify issues

3. **Generate Report**
   - Create validation report
   - List errors found
   - Recommend fixes
   - Suggest additional schemas

## What to Tell the User

- "What pages do you want to check for structured data?"
- "What schema types do you need?"
- Validate structured data
- "Structured data check complete! Status: [valid/invalid]. Errors: [list]. Recommendations: [list]"
