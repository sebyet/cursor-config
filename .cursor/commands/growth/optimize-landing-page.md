# Optimize Landing Page

Optimize landing pages for conversion, performance, and user experience.

## What This Command Does

Optimizes landing pages by:
- Analyzing current performance
- Identifying conversion barriers
- Suggesting improvements
- Optimizing copy and design
- Creating optimization plan

## How It Works

1. **Understand Landing Page**
   - Ask about landing page to optimize
   - Get current performance data
   - Understand conversion goals
   - Identify target audience

2. **Analyze Landing Page**
   - Review current design and copy
   - Identify conversion barriers
   - Check performance metrics
   - Find optimization opportunities

3. **Generate Optimization Plan**
   - Create recommendations
   - Prioritize improvements
   - Suggest specific changes
   - Provide implementation steps

## What to Tell the User

- "What landing page do you want to optimize?"
- "What are your conversion goals?"
- Analyze and generate optimization plan
- "Landing page optimization complete! Recommendations: [list]. Priority improvements: [list]"
