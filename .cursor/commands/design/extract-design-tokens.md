# Extract Design Tokens

Extract colors, spacing, typography, and other design tokens from design files.

## What This Command Does

Extracts design tokens by:
- Identifying colors and color system
- Extracting spacing values
- Documenting typography scale
- Finding border radius, shadows, etc.
- Generating token configuration

## How It Works

1. **Gather Design Files**
   - Ask for design files (Figma, Sketch, etc.)
   - Identify what tokens to extract
   - Understand design system structure

2. **Extract Tokens**
   - Identify color palette
   - Extract spacing scale
   - Document typography
   - Find other design tokens

3. **Generate Token Config**
   - Create token configuration
   - Format for CSS variables or design system
   - Document usage
   - Create implementation guide

## What to Tell the User

- "What design files do you have?"
- "What tokens do you want to extract?"
- Extract design tokens
- "Design tokens extracted! Configuration: [details]. Ready to implement."
