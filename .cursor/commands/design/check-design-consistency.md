# Check Design Consistency

Verify design system compliance across components and pages.

## What This Command Does

Checks design consistency by:
- Auditing design system usage
- Identifying inconsistencies
- Checking component variants
- Verifying spacing and typography
- Ensuring brand guidelines are followed

## How It Works

1. **Define Scope**
   - Ask what to check (components, pages, features)
   - Understand design system
   - Identify consistency rules

2. **Audit Design**
   - Review components for consistency
   - Check design token usage
   - Verify spacing patterns
   - Check typography hierarchy
   - Identify violations

3. **Generate Report**
   - List inconsistencies found
   - Prioritize issues
   - Suggest fixes
   - Provide recommendations

## What to Tell the User

- "What do you want to check for consistency?"
- "What's your design system?"
- Audit design consistency
- "Consistency check complete! Issues found: [list]. Recommendations: [list]"
