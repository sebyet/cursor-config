# Onboarding Documentation

Create developer onboarding documentation for new team members.

## What This Command Does

Creates onboarding docs by:
- Documenting setup procedures
- Creating getting started guides
- Documenting development workflow
- Including project overview
- Providing learning resources

## How It Works

1. **Understand Onboarding Needs**
   - Ask about onboarding scope
   - Identify setup requirements
   - Understand development workflow
   - Determine documentation needs

2. **Create Onboarding Documentation**
   - Document setup steps
   - Create getting started guide
   - Document workflow
   - Include resources

3. **Save and Organize**
   - Save documentation
   - Organize in docs
   - Create index
   - Update onboarding guide

## What to Tell the User

- "What should we include in onboarding?"
- "What setup steps are needed?"
- Create onboarding documentation
- "Onboarding documentation created! Sections: [count]. Guide: [path]. Resources: [included]"
