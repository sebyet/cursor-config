# Local Dev Setup

Set up local development environment with Docker, environment variables, and database seeding.

## What This Command Does

Sets up local development by:
- Configuring Docker setup
- Setting up environment variables
- Configuring database seeding
- Setting up development tools
- Creating development documentation

## How It Works

1. **Understand Dev Environment Needs**
   - Ask about development requirements
   - Identify services needed
   - Understand database setup
   - Get environment preferences

2. **Set Up Docker**
   - Create Docker configuration
   - Set up service containers
   - Configure database container
   - Set up development volumes
   - Configure networking

3. **Configure Environment Variables**
   - Create environment files
   - Set up variable templates
   - Configure secrets management
   - Add development defaults
   - Document variables

4. **Set Up Database Seeding**
   - Create seed scripts
   - Set up test data
   - Configure migrations
   - Add seed utilities
   - Document seed process

5. **Create Development Documentation**
   - Document setup steps
   - Create development guide
   - Add troubleshooting tips
   - Document common tasks

## What to Tell the User

- "What development environment do you need?"
- "What services should be included?"
- Set up local development
- "Local dev setup complete! Docker: [configured]. Environment: [setup]. Database: [seeded]. Start with: [command]"
