# Backup Database

Set up database backup strategies and automated backups.

## What This Command Does

Sets up database backups by:
- Configuring backup schedules
- Setting up automated backups
- Configuring backup retention
- Testing backup restoration
- Documenting backup procedures

## How It Works

1. **Understand Backup Needs**
   - Ask about database type
   - Identify backup frequency
   - Understand retention requirements
   - Determine backup storage

2. **Design Backup Strategy**
   - Configure backup schedule
   - Set up automated backups
   - Configure retention policy
   - Plan backup storage

3. **Generate Configuration**
   - Create backup configuration
   - Set up automation
   - Configure retention
   - Document procedures

## What to Tell the User

- "What database are you using? (PostgreSQL, MySQL, MongoDB, etc.)"
- "How often should we backup? (daily, hourly, etc.)"
- Generate backup configuration
- "Database backup setup complete! Schedule: [frequency]. Retention: [policy]. Storage: [configured]"
