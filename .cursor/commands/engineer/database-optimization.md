# Database Optimization

Optimize database performance through query optimization, indexing, and tuning.

## What This Command Does

Optimizes databases by:
- Analyzing slow queries
- Creating optimal indexes
- Optimizing query performance
- Tuning database configuration
- Generating optimization recommendations

## How It Works

1. **Understand Optimization Needs**
   - Ask about performance issues
   - Identify slow queries
   - Understand database type
   - Determine optimization scope

2. **Analyze and Optimize**
   - Analyze query performance
   - Identify optimization opportunities
   - Create indexes
   - Optimize queries

3. **Verify and Document**
   - Verify improvements
   - Measure performance gains
   - Document optimizations
   - Generate recommendations

## What to Tell the User

- "What performance issues are you experiencing?"
- "What database are you using?"
- Analyze and optimize database
- "Database optimization complete! Slow queries: [fixed]. Indexes: [created]. Performance: [improved]"
