# Write API Docs

Generate API documentation (OpenAPI/Swagger) from code or specifications.

## What This Command Does

Generates API documentation by:
- Creating OpenAPI/Swagger specifications
- Documenting API endpoints
- Generating API reference docs
- Creating interactive API docs
- Documenting request/response schemas

## How It Works

1. **Understand API Documentation Needs**
   - Ask about API to document
   - Identify documentation format
   - Understand API structure
   - Determine documentation scope

2. **Generate API Documentation**
   - Extract or create API spec
   - Document endpoints
   - Generate reference docs
   - Create interactive docs

3. **Save and Publish**
   - Save documentation
   - Set up hosting
   - Update API index
   - Document setup

## What to Tell the User

- "What API should we document?"
- "What format? (OpenAPI, Swagger, etc.)"
- Generate API documentation
- "API documentation created! Endpoints: [count]. Format: [type]. Interactive docs: [generated]"
