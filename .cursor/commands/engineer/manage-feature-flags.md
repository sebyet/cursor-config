# Manage Feature Flags

Manage feature flags beyond initial setup - toggle, configure, and analyze feature flags.

## What This Command Does

Manages feature flags by:
- Toggling feature flags on/off
- Configuring flag targeting (users, percentages, segments)
- Analyzing flag usage and impact
- Managing flag lifecycle
- Cleaning up unused flags

## How It Works

1. **Understand Feature Flag Needs**
   - Ask about flag to manage
   - Identify action (toggle, configure, analyze)
   - Understand targeting requirements
   - Determine flag scope

2. **Manage Feature Flag**
   - Toggle flag state
   - Configure targeting
   - Analyze usage
   - Update flag configuration

3. **Document and Verify**
   - Document flag changes
   - Verify flag state
   - Monitor flag impact
   - Plan next steps

## What to Tell the User

- "What feature flag do you want to manage?"
- "What action? (toggle, configure, analyze, cleanup)"
- Manage feature flag and verify
- "Feature flag managed! Flag: [name]. State: [on/off]. Targeting: [configured]"
