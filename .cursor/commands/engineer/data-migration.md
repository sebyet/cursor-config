# Data Migration

Perform data migration beyond schema changes - move, transform, and migrate data.

## What This Command Does

Performs data migration by:
- Planning data migration strategy
- Creating migration scripts
- Executing data transformation
- Validating migrated data
- Documenting migration process

## How It Works

1. **Understand Migration Needs**
   - Ask about data to migrate
   - Identify source and target
   - Understand transformation requirements
   - Determine migration scope

2. **Design Migration**
   - Create migration plan
   - Design transformation logic
   - Plan validation steps
   - Set up rollback strategy

3. **Execute Migration**
   - Run migration scripts
   - Transform data
   - Validate results
   - Document process

## What to Tell the User

- "What data needs to be migrated?"
- "What is the source and target?"
- Execute data migration and validate
- "Data migration complete! Records: [count]. Validation: [passed]. Rollback: [available]"
