# Create Troubleshooting Guide

Create troubleshooting guides for common issues and problems.

## What This Command Does

Creates troubleshooting guides by:
- Documenting common issues
- Creating step-by-step solutions
- Including diagnostic steps
- Providing workarounds
- Organizing by problem type

## How It Works

1. **Understand Troubleshooting Needs**
   - Ask about issues to document
   - Identify problem categories
   - Understand solution approaches
   - Determine guide structure

2. **Create Troubleshooting Guide**
   - Document common issues
   - Create solutions
   - Include diagnostics
   - Organize by category

3. **Save and Organize**
   - Save guide file
   - Organize in documentation
   - Create index
   - Document guide

## What to Tell the User

- "What issues should we document?"
- "What problem categories do you have?"
- Create troubleshooting guide
- "Troubleshooting guide created! Issues: [count]. Categories: [list]. Guide: [path]"
