# Fix Git Issues

Fixes Git problems like merge conflicts, uncommitted changes, branch issues, and repository problems. Gets your Git repository back to a clean, working state.

1. **Identify Git problems**
   - Check Git status for uncommitted changes
   - Check for merge conflicts
   - Check branch state
   - Check repository connection
   - Check for rebase issues
   - Tell the user what you found

2. **Fix Git issues**
   - **If uncommitted changes:** Help commit or stash them
   - **If merge conflicts:** Resolve conflicts automatically or guide resolution
   - **If branch issues:** Fix branch state (reset, clean, etc.)
   - **If repository issues:** Fix connection or repository state
   - **If rebase issues:** Abort rebase or resolve conflicts
   - Show what you're fixing

3. **Verify Git is clean**
   - Check that Git status is clean
   - Verify no conflicts remain
   - Confirm repository is in good state
   - Tell them when it's fixed!

## What to Tell the User

- Explain you're checking Git status
- List what problems you found (e.g., "Found uncommitted changes and a merge conflict...")
- Fix issues one by one, explaining what you're doing
- Show progress as you fix each issue
- Verify Git is clean after fixes
- Tell them when all Git issues are resolved
