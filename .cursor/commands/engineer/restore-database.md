# Restore Database

Restore database from backups when needed.

## What This Command Does

Restores databases by:
- Identifying backup to restore
- Restoring database from backup
- Verifying restore success
- Documenting restore process
- Notifying stakeholders

## How It Works

1. **Understand Restore Needs**
   - Ask about restore reason
   - Identify backup to restore
   - Understand restore scope
   - Determine restore target

2. **Execute Restore**
   - Restore from backup
   - Verify restore success
   - Check data integrity
   - Confirm restore completion

3. **Document and Notify**
   - Document restore process
   - Update backup history
   - Notify team
   - Plan follow-up actions

## What to Tell the User

- "What backup should we restore from?"
- "What is the restore reason? (disaster recovery, data corruption, etc.)"
- Execute restore and verify
- "Database restore complete! Backup: [date]. Status: [verified]. Data integrity: [checked]"
