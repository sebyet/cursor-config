# Analyze A/B Test

Analyze A/B test results to determine statistical significance and winning variant.

## What This Command Does

Analyzes A/B tests by:
- Calculating statistical significance
- Comparing variant performance
- Identifying winning variant
- Analyzing segment differences
- Generating recommendations

## How It Works

1. **Gather Test Data**
   - Ask for test results
   - Get variant metrics
   - Understand test setup
   - Identify primary metric

2. **Analyze Results**
   - Calculate statistical significance
   - Compare variant performance
   - Analyze by segments
   - Check for anomalies

3. **Generate Analysis**
   - Determine winning variant
   - Assess confidence level
   - Identify insights
   - Recommend next steps

## What to Tell the User

- "What A/B test do you want to analyze?"
- "What are the results for each variant?"
- Analyze test results
- "A/B test analysis complete! Winner: [variant] with [confidence]% confidence. Insights: [list]"
