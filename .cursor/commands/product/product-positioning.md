# Product Positioning

Define product positioning statement, value proposition, and messaging framework.

## What This Command Does

Defines positioning by:
- Creating positioning statement
- Defining value proposition
- Developing messaging framework
- Identifying target audience
- Analyzing competitive positioning
- Generating positioning strategy

## How It Works

1. **Understand Positioning Goals**
   - Ask about positioning objectives
   - Identify target audience
   - Understand market context
   - Get context on goals

2. **Create Positioning Statement**
   - Define target audience
   - Identify key benefits
   - Create positioning statement
   - Refine positioning
   - Document positioning

3. **Define Value Proposition**
   - Identify unique value
   - Define value proposition
   - Create value messaging
   - Test value proposition
   - Refine messaging

4. **Develop Messaging Framework**
   - Create key messages
   - Develop messaging hierarchy
   - Create messaging guidelines
   - Set up messaging templates
   - Document messaging framework

## What to Tell the User

- "What product positioning do you want to define?"
- "Who is your target audience?"
- Create positioning strategy
- "Product positioning complete! Statement: [details]. Value proposition: [defined]. Messaging: [framework]"
