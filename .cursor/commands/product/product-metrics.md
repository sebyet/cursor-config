# Product Metrics

Define product metrics framework including North Star metric, KPIs, and metrics tracking.

## What This Command Does

Defines product metrics by:
- Defining North Star metric
- Creating KPI framework
- Designing metrics dashboard
- Setting up metrics tracking
- Analyzing metrics trends
- Generating metrics insights

## How It Works

1. **Understand Metrics Goals**
   - Ask about product goals
   - Identify key metrics
   - Understand success criteria
   - Get context on objectives

2. **Define North Star Metric**
   - Identify North Star metric
   - Define metric calculation
   - Set metric targets
   - Plan metric tracking
   - Document metric definition

3. **Create KPI Framework**
   - Define key KPIs
   - Set KPI targets
   - Create KPI hierarchy
   - Plan KPI tracking
   - Document KPI framework

4. **Design Metrics Dashboard**
   - Design dashboard layout
   - Select key metrics
   - Create visualizations
   - Set up dashboard
   - Configure alerts

5. **Set Up Metrics Tracking**
   - Implement metrics tracking
   - Configure data collection
   - Set up metrics pipeline
   - Create metrics reports
   - Document tracking setup

## What to Tell the User

- "What product metrics do you want to define?"
- "What's your North Star metric?"
- Define metrics framework
- "Product metrics framework complete! North Star: [metric]. KPIs: [count]. Dashboard: [configured]"
