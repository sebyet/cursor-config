# Stakeholder Communication

Create stakeholder communication templates, status updates, and reports.

## What This Command Does

Manages stakeholder communication by:
- Creating communication templates
- Generating status updates
- Creating stakeholder reports
- Planning communication cadence
- Managing stakeholder feedback
- Tracking communication effectiveness

## How It Works

1. **Understand Communication Needs**
   - Ask about stakeholders
   - Identify communication goals
   - Understand update frequency
   - Get context on needs

2. **Create Communication Templates**
   - Design status update template
   - Create report templates
   - Set up email templates
   - Create presentation templates
   - Document templates

3. **Generate Status Updates**
   - Create status update
   - Include key metrics
   - Highlight achievements
   - Identify blockers
   - Generate update content

4. **Create Stakeholder Reports**
   - Generate progress report
   - Include metrics dashboard
   - Add insights
   - Create recommendations
   - Format report

## What to Tell the User

- "What stakeholder communication do you need?"
- "Who are your stakeholders?"
- Create communication materials
- "Stakeholder communication ready! Templates: [created]. Status update: [generated]. Reports: [available]"
