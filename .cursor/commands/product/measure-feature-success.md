# Measure Feature Success

Measure feature success using defined metrics and KPIs.

## What This Command Does

Measures feature success by:
- Defining success metrics
- Measuring feature performance
- Comparing against goals
- Analyzing success factors
- Generating success reports

## How It Works

1. **Understand Success Measurement Needs**
   - Ask about features to measure
   - Identify success metrics
   - Understand goals
   - Determine measurement period

2. **Measure Feature Success**
   - Collect metrics
   - Measure performance
   - Compare to goals
   - Analyze factors

3. **Generate Report**
   - Create success report
   - Identify achievements
   - Provide insights
   - Document findings

## What to Tell the User

- "What features should we measure?"
- "What are your success metrics and goals?"
- Measure feature success and generate report
- "Feature success measurement complete! Features: [count]. Metrics: [measured]. Success: [assessment]"
