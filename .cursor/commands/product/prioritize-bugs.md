# Prioritize Bugs

Prioritize bugs using frameworks and criteria for efficient bug resolution.

## What This Command Does

Prioritizes bugs by:
- Applying prioritization frameworks
- Assessing bug severity and impact
- Considering user impact
- Creating prioritized bug list
- Generating resolution recommendations

## How It Works

1. **Understand Bug Prioritization Needs**
   - Ask about bugs to prioritize
   - Identify prioritization criteria
   - Understand user impact
   - Determine framework preference

2. **Prioritize Bugs**
   - Assess severity
   - Evaluate impact
   - Apply framework
   - Create priority list

3. **Generate Report**
   - Create prioritized list
   - Provide recommendations
   - Document priorities
   - Plan resolution

## What to Tell the User

- "What bugs should we prioritize?"
- "What prioritization framework? (severity, impact, user count, etc.)"
- Prioritize bugs and generate report
- "Bug prioritization complete! Bugs: [count]. Priorities: [assigned]. Recommendations: [list]"
