# Competitive Analysis

Analyze competitors' features, positioning, and market strategies to inform product decisions.

## What This Command Does

Analyzes competitors by:
- Analyzing competitor features
- Comparing market positioning
- Benchmarking against competitors
- Identifying competitive advantages
- Performing SWOT analysis
- Generating competitive insights

## How It Works

1. **Understand Analysis Goals**
   - Ask about competitors to analyze
   - Identify analysis objectives
   - Understand market context
   - Get context on goals

2. **Research Competitors**
   - Identify key competitors
   - Research competitor features
   - Analyze competitor positioning
   - Review competitor strategies
   - Gather competitor data

3. **Perform Analysis**
   - Compare features
   - Analyze positioning
   - Benchmark capabilities
   - Identify gaps
   - Find opportunities

4. **Generate Insights**
   - Create competitive matrix
   - Perform SWOT analysis
   - Identify advantages
   - Recommend strategies
   - Generate competitive report

## What to Tell the User

- "What competitors do you want to analyze?"
- "What's your analysis goal?"
- Research and analyze competitors
- "Competitive analysis complete! Competitors: [count]. Key insights: [list]. Recommendations: [details]"
