# User Research

Synthesize user research including interviews, surveys, and usability testing to generate insights.

## What This Command Does

Synthesizes user research by:
- Analyzing user interviews
- Synthesizing survey data
- Reviewing usability testing
- Identifying user insights
- Generating research findings
- Creating action plans

## How It Works

1. **Understand Research Context**
   - Ask about research conducted
   - Identify research methods
   - Understand research goals
   - Get context on findings

2. **Synthesize Research Data**
   - Analyze interview transcripts
   - Synthesize survey responses
   - Review usability findings
   - Identify patterns
   - Extract insights

3. **Generate Insights**
   - Create user insights
   - Identify pain points
   - Find opportunities
   - Generate recommendations
   - Create research summary

4. **Create Action Plan**
   - Prioritize insights
   - Create action items
   - Plan implementation
   - Set success metrics
   - Generate research report

## What to Tell the User

- "What user research do you want to synthesize?"
- "What research methods were used?"
- Synthesize research
- "User research synthesis complete! Insights: [count]. Key findings: [list]. Action plan: [details]"
