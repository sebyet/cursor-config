# Retrospective

Facilitate sprint or iteration retrospectives to identify improvements.

## What This Command Does

Facilitates retrospectives by:
- Structuring retrospective format
- Collecting team feedback
- Identifying what went well
- Finding improvement opportunities
- Creating action items

## How It Works

1. **Understand Retrospective Needs**
   - Ask about retrospective scope
   - Identify time period
   - Understand team size
   - Determine format preferences

2. **Facilitate Retrospective**
   - Structure format
   - Collect feedback
   - Identify patterns
   - Create action items

3. **Generate Report**
   - Create retrospective report
   - Document findings
   - List action items
   - Plan follow-up

## What to Tell the User

- "What is the retrospective scope? (sprint, iteration, project)"
- "What time period are we reviewing?"
- Facilitate retrospective and generate report
- "Retrospective complete! Period: [scope]. Findings: [summary]. Action items: [count]"
