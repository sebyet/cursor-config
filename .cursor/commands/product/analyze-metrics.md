# Analyze Metrics

Analyze key metrics and trends to understand performance and identify opportunities.

## What This Command Does

Analyzes metrics by:
- Reviewing key performance indicators
- Identifying trends and patterns
- Comparing time periods
- Finding anomalies
- Generating insights and recommendations

## How It Works

1. **Define Metrics to Analyze**
   - Ask about key metrics (conversion, retention, engagement, etc.)
   - Understand business context
   - Identify time period
   - Get access to data

2. **Analyze Data**
   - Review metric values
   - Calculate trends
   - Compare periods
   - Identify patterns

3. **Generate Insights**
   - Summarize findings
   - Highlight key trends
   - Identify opportunities
   - Recommend actions

## What to Tell the User

- "What metrics do you want to analyze?"
- "What time period?"
- Analyze metrics and trends
- "Metrics analysis complete! Key findings: [summary]. Trends: [list]. Recommendations: [list]"
