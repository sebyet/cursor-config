# Create Roadmap

Generate roadmap documents that align features, timelines, and business goals.

## What This Command Does

Creates comprehensive roadmaps that include:
- Strategic themes and goals
- Feature timeline and milestones
- Dependencies and blockers
- Resource requirements
- Success metrics per phase

## How It Works

1. **Gather Information**
   - Ask about strategic goals
   - Get prioritized feature list
   - Understand timeline constraints
   - Identify dependencies

2. **Structure Roadmap**
   - Organize by quarters/periods
   - Group related features
   - Show dependencies
   - Include milestones

3. **Generate Roadmap**
   - Create visual timeline
   - Show feature groupings
   - Highlight key milestones
   - Include success metrics

## What to Tell the User

- "What's your strategic goal for this roadmap?"
- "What features need to be included?"
- "What's your timeline? (quarters, months, etc.)"
- Generate roadmap document
- "Roadmap created! Review timeline and adjust as needed."
