# Identify Insights

Find insights from data by analyzing patterns, anomalies, and correlations.

## What This Command Does

Identifies data insights by:
- Analyzing data patterns
- Finding anomalies
- Discovering correlations
- Generating hypotheses
- Creating actionable recommendations

## How It Works

1. **Understand Data Context**
   - Ask about data sources
   - Identify key metrics
   - Understand business questions
   - Get access to data

2. **Analyze Data**
   - Look for patterns
   - Identify anomalies
   - Find correlations
   - Generate hypotheses

3. **Generate Insights**
   - Summarize findings
   - Highlight key insights
   - Create hypotheses
   - Recommend actions

## What to Tell the User

- "What data do you want to analyze?"
- "What questions are you trying to answer?"
- Analyze data and identify insights
- "Insights identified! Key findings: [list]. Hypotheses: [list]. Recommended actions: [list]"
