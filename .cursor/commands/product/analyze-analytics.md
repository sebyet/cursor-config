# Analyze Analytics

Interpret analytics data to understand user behavior and identify opportunities.

## What This Command Does

Analyzes analytics data to:
- Identify user behavior patterns
- Find drop-off points and friction
- Track key metrics and trends
- Generate insights and recommendations
- Create actionable reports

## How It Works

1. **Understand Analytics Context**
   - Ask about analytics platform (Google Analytics, Mixpanel, etc.)
   - Identify key metrics to analyze
   - Understand business goals
   - Get access to data or reports

2. **Analyze Data**
   - Review key metrics
   - Identify trends and patterns
   - Find anomalies or issues
   - Compare time periods

3. **Generate Insights**
   - Create summary of findings
   - Highlight key insights
   - Identify opportunities
   - Recommend actions

## What to Tell the User

- "What analytics platform are you using?"
- "What metrics do you want to analyze?"
- "What's your time period?"
- Analyze data and generate insights
- "Analytics analysis complete! Key insights: [list]. Recommended actions: [list]"
