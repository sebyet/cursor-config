# Analyze Feature Adoption

Analyze feature adoption rates and user engagement with features.

## What This Command Does

Analyzes feature adoption by:
- Measuring feature usage rates
- Identifying adoption trends
- Analyzing user segments
- Comparing adoption across features
- Generating adoption recommendations

## How It Works

1. **Understand Adoption Analysis Needs**
   - Ask about features to analyze
   - Identify time period
   - Understand user segments
   - Determine analysis scope

2. **Analyze Feature Adoption**
   - Measure usage rates
   - Analyze trends
   - Segment users
   - Compare features

3. **Generate Report**
   - Create adoption report
   - Identify insights
   - Provide recommendations
   - Document findings

## What to Tell the User

- "What features should we analyze?"
- "What time period should we analyze?"
- Analyze feature adoption and generate report
- "Feature adoption analysis complete! Features: [count]. Adoption rates: [summary]. Insights: [list]"
