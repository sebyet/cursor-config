# Create Dashboard

Generate analytics dashboard queries and configurations for tracking KPIs.

## What This Command Does

Creates dashboard configurations that include:
- Key metrics to track
- Query definitions
- Visualization recommendations
- Update frequency
- Alert thresholds

## How It Works

1. **Understand Dashboard Needs**
   - Ask about key metrics to track
   - Understand business goals
   - Identify data sources
   - Determine update frequency

2. **Design Dashboard**
   - Select key metrics
   - Create query definitions
   - Recommend visualizations
   - Set up alerts

3. **Generate Dashboard Config**
   - Create dashboard structure
   - Include queries and formulas
   - Add visualization specs
   - Document setup instructions

## What to Tell the User

- "What metrics do you want to track in this dashboard?"
- "What's the purpose of this dashboard?"
- Generate dashboard configuration
- "Dashboard configuration created! Set up with these queries: [list]"
