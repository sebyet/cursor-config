# Track Conversion

Set up conversion tracking to measure user actions and funnel performance.

## What This Command Does

Sets up conversion tracking by:
- Defining conversion events
- Creating tracking implementation
- Setting up funnel analysis
- Configuring attribution
- Creating measurement plan

## How It Works

1. **Define Conversion Goals**
   - Ask about conversion events to track
   - Understand user journey
   - Identify key actions
   - Determine attribution model

2. **Design Tracking**
   - Map conversion events
   - Create tracking plan
   - Define funnel stages
   - Set up attribution

3. **Generate Implementation**
   - Create tracking code/config
   - Document setup steps
   - Add validation checks
   - Create measurement dashboard

## What to Tell the User

- "What conversion events do you want to track?"
- "What's the user journey to conversion?"
- Generate tracking implementation
- "Conversion tracking setup complete! Track these events: [list]. Implementation: [details]"
