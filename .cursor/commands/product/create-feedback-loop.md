# Create Feedback Loop

Set up user feedback collection system for continuous improvement.

## What This Command Does

Creates feedback loops by:
- Setting up feedback collection (surveys, in-app, etc.)
- Configuring feedback channels
- Creating feedback workflows
- Setting up feedback analysis
- Documenting feedback process

## How It Works

1. **Understand Feedback Needs**
   - Ask about feedback channels
   - Identify feedback types
   - Understand collection methods
   - Determine analysis needs

2. **Design Feedback Loop**
   - Choose feedback channels
   - Set up collection
   - Plan analysis
   - Design workflows

3. **Generate Configuration**
   - Create feedback config
   - Set up collection
   - Configure analysis
   - Document setup steps

## What to Tell the User

- "What feedback channels should we use? (surveys, in-app, email, etc.)"
- "What types of feedback should we collect?"
- Generate feedback loop configuration
- "Feedback loop setup complete! Channels: [list]. Collection: [configured]. Analysis: [setup]"
