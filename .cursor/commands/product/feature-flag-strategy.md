# Feature Flag Strategy

Plan feature flag rollout strategy including gradual rollouts, targeting, and flag management.

## What This Command Does

Plans feature flags by:
- Creating flag rollout strategy
- Planning gradual rollouts
- Setting up flag targeting
- Managing flag lifecycle
- Configuring flag rules
- Tracking flag performance

## How It Works

1. **Understand Flag Goals**
   - Ask about feature flags
   - Identify rollout objectives
   - Understand targeting needs
   - Get context on goals

2. **Create Rollout Strategy**
   - Plan rollout phases
   - Define rollout percentages
   - Set rollout timeline
   - Plan rollback strategy
   - Create rollout plan

3. **Set Up Flag Targeting**
   - Configure user targeting
   - Set up segment targeting
   - Configure geographic targeting
   - Add targeting rules
   - Test targeting logic

4. **Manage Flag Lifecycle**
   - Plan flag creation
   - Set flag expiration
   - Plan flag removal
   - Document flag usage
   - Track flag status

## What to Tell the User

- "What feature flags do you want to plan?"
- "What's your rollout strategy?"
- Create feature flag strategy
- "Feature flag strategy complete! Rollout plan: [details]. Targeting: [configured]. Management: [setup]"
