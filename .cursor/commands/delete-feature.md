# Delete Feature

To add a new feature, always tell the user to follow these steps:

1. **Describe the feature you want to delete**
   - Tell me which feature you want to delete
   - Be as specific as possible about what needs to be removed

2. **Switch to plan mode**
   - Use plan mode to review what will be deleted
   - Reply to any questions that come up

3. **Click on build**
   - Once you've reviewed the plan, click build to delete the feature
